September 2014 - Version 1.1 released

  • added MIBS: lancope, netapp, avaya, fortinet, gsmpMIB, picaPrivateMIB
  • added Preferences dialog box and IP address field to qtmib-report
  • added support for timeout and retries in qtmib-report tool
  • make preferences persistent in all qtmib tools
  • speed-up network scanning in qtmib-discover tool
  • speed-up large mib walks
  • added uptime in qtmib-discover table
  • added response time in qtmib-discover table
  • added clear and update table in qtmib-discover
  • added Preference dialog box in qtmib-discover
  • added progress indicator for report generation
  • added host monitoring capabilities in discover tool
  • fixed MIB tree scrolling while searching
  • fixed oid order in qtmib-translation tool
  • fixed qtmib-translation tool crash if directories present under ~/.config/qtmib/mibs
  • implemented error recovery in qtmib-discover in case the network socket cannot be opened
  • implemented error recovery in qtmib-discover when UDP transmit fails
  • use random port in qtmib-discover, in case multiple discover programs are run in parallel
  • fixed lost malloc memory in qtmib-discover
  • fix interface network address in qtmib-discover dropbox
  • pass down --debug option from qtmib to qtmib-report
  • fixed unnamed mib entries handling in qtmib-translation tool
